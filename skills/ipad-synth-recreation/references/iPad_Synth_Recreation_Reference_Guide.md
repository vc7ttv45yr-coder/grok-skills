# iPad Synth Recreation Reference Guide & Custom Patch Templates

Matching Synthesizer Sounds from Audio Analysis using Logic Pro, Buttersynth & More

## 1. Introduction & Workflow
This guide helps recreate synthesizer sounds detected in audio files using the audio-analyzer skill. It turns MIR output into actionable patch settings for iPad synths.

Recommended Workflow:
1. Run analysis with audio-analyzer.
2. Review output (spectral_centroid_hz, harmonic_ratio, percussive_ratio, etc.).
3. Use templates and mappings below.
4. Load in Logic Pro Alchemy or Buttersynth and ear-match.
